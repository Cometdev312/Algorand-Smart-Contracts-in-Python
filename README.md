# PyTeal

Python Language Binding for Teal


### Development Setup

Setup venv (one time):
 * `python3 -m venv venv`


Active venv:
 * `. venv/bin/activate.fish` (if your shell is fish)
 * `. venv/bin/activate` (if your shell is bash/zsh)


Pip install pyteal in aditable state
 * `pip install -e .`
 
Type checking using mypy
* `mypy pyteal`
